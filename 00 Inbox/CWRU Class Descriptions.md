## Cleaner version

The goal of this class is to teach algorithmic thinking through basic Python programming. The course is highly hands-on: students follow along by typing code examples and completing guided exercises, then apply those skills independently in their own assignments.

## A slightly more polished version for a syllabus / website

This course introduces business students to algorithmic thinking through beginner-friendly Python programming. It is designed for students with little to no technical background and emphasizes hands-on learning through live coding, guided exercises, and independent practice.
