Defintions:
- `<c>` - any character

Movement
- `h`
- `j`
- `k`
- `l`
- `gg` - go to the start of a document
- `G` - go to the end of a document
-  - go to start of a block
- - go to the end of a block
- `M` - go to the middle of the screen
- `zz` - move the line with the cursor to the middle of the screen
## Finding Text In A Line
- `f<c>` - find the next occurrence of `<c>`
- `;` - repeat the last find command
- `,` - move to the previous found target
- `F<c>` - find the prev occurrence of `<c>`

## Other Tricks
- `==` - to have Vim fix the indentation
- `ZZ` - to close Vim
- `ZQ` - close vim without saving
